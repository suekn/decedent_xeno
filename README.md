# README

## Donor-reactive immune repertoire tracking in a pig-to-human decedent kidney transplant reveals roles for T cells and innate immune cells in xenograft rejection
**Single Cell and TCR Sequencing Analysis**

## Description
This repository contains the analysis code and associated scripts for the paper titled "Donor-reactive immune repertoire tracking in a pig-to-human decedent kidney transplant reveals roles for T cells and innate immune cells in xenograft rejection". The work focuses on leveraging single-cell RNA sequencing and TCR sequencing to study immune cell dynamics in xenotransplantation.

## Repository Structure
- **integrated2**: Contains code for scRNAseq analysis

## Installation
Clone this repository to your local machine:
```bash
git clone https://github.com/suekn/decedent_xeno.git
```

Install required R and Python packages:
```bash
# R dependencies
install.packages(c("Seurat", "SingleCellExperiment", "tidyverse", "ggplot2", "ggalluvial"))
```
